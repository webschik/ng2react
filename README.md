# ng2react

This module will help you to migrate [Angular 1](https://angularjs.org/) project
to [React.js](https://reactjs.org/)

## Usage
```shell
npm i -D ng2react
```

### CLI
```shell
npx ng2react
```

### via API
```js
import {createReactComponent} from 'ng2react';
```
